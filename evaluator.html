Phase 1: Gate-Keeping Criteria

Is the question package complete? Required: statement, answer key, explanation. For MCQ: 4-5 options. For Open-ended: model answer/rubric. FAIL if missing components, truncated content, broken formatting, or placeholders.
Is the question factually and conceptually correct? FAIL if contains errors, wrong answer key, logical contradictions, or multiple defensible correct answers.

Phase 2: Quality Assessment

0-1: Obviously wrong distractors, no explanations. 2-3: Plausible but generic distractors, superficial explanations. 4-5: Specific misconceptions, good explanations. 6: Excellence - well-documented misconceptions, explicit naming of error patterns.
0-1: No model answer/rubric or too vague. 2-3: Some guidance but incomplete, unclear quality levels. 4-5: Detailed model answer/rubric with clear expectations. 6: Excellence - comprehensive model answer or complete rubric with specific criteria and objective scoring.
